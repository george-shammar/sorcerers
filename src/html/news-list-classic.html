SORCERERS Receives The Next Step Microgrant From Filecoin/IPFS

June 11, 2022

The entire team at SORCERERS is happy to be a beneficiary of the Filecoin/IPFS Next Step Microgrant. Designed to support taking the next step after building an initial prototype with Filecoin, SORCERERS uses the Interplanetary File System to store all game assets decentrally.

The project was a Hackathon participant sponsored by Filecoin/IPFS and hosted by Encode Club in late 2021. Initially known as Mage Wars, the project won the first prize in the "Bridging Challenge" category and the Runner-up Prize in the "Unleash Challenge" category. This partly qualified the project for the Encode/Filecoin Accelerator Program.
